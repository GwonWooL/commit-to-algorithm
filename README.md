# 💻 코딩테스트 스터디

이 레포는 코딩테스트 준비를 위한 스터디 기록용 저장소입니다.  
문제 풀이, 코드 리뷰, 알고리즘 정리를 통해 실전 감각을 기르는 것을 목표로 합니다.

---

## 📅 진행 방식

- 매일 최소 1문제 이상 풀이 업로드
- 개인 Fork → PR → 코드 리뷰 → Merge
- 코드 리뷰는 풀이 접근, 시간/공간 복잡도, 가독성 중심으로 진행
